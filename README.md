# Web Turntable Viewer Widget (Beta)

> ⚠️ まだバージョン1未満のベータ版です。仕様が予告なく変更される可能性があります。

模型や3DCGモデルを360度回転させたターンテーブル動画をWebブラウザ上でドラッグ操作して自由な角度から鑑賞できるウィジェットスクリプトです。
Vimeoの動画埋め込みをラップするもので、動画のアップロード・公開にはVimeoを利用します。
